    Image caption generation is a task in computer vision and natural language processing where the goal is to generate a textual description (caption) for an input image. A compelling application of deep learning, involves the development of models capable of providing descriptive and meaningful textual descriptions for input images where the CNN-LSTM combination is a state-of-the-art deep learning model that uses neural networks' power to enable automated picture captioning.
